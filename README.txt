
# Enhanced CNA Student Portal Features

This bundle includes the following additional features:

1. **Student Progress History Tracker**: Tracks performance over time with visual charts.
2. **Personalized Goal Setting + Reminders**: Lets students define goals and receive reminders.
3. **Document Upload & Resource Library**: Upload area for assignments, documents, and CNA training resources.
4. **Support & Contact Center**: Request help and view community resources and emergency contacts.
5. **AI Study Assistant**: GPT-powered chatbot to assist with studying, CNA preparation, and personalized guidance.
6. **Exportable Progress Report**: Automatically generates downloadable PDFs or CSVs with performance summaries.

Each feature includes placeholder files to be used in integration with the full Streamlit dashboard.
